Sistema de Luces Tally para Producción IP

Control Visual Inalámbrico con Seguimiento Pan/Tilt Integrado

Autores: Christian Imbaquingo y Barila Sota | Centro Académico: IES Pacífico

1. El Reto

En la producción audiovisual IP, los operadores necesitan saber en tiempo real qué cámara está en Program o Preview. El objetivo es democratizar esta tecnología con una solución open-source, inalámbrica y de baja latencia frente a los costosos sistemas comerciales.

2. Arquitectura de Hardware

  • Unidad de Captura: Módulos ESP32-CAM que procesan las señales de red y activan los LEDs Tally.
  • Mecánica: Sistema Pan/Tilt impulsado por servomotores SG90 para encuadre remoto.
  • Chasis a Medida: Diseño de soportes en impresión 3D sin adhesivos permanentes, con ajuste por fricción (Dual Lock / Masilla Inofix).
  • Unidad Central: Raspberry Pi para la orquestación del sistema.

3. Software y Comunicaciones

  • Conectividad: Infraestructura Wi-Fi local para máxima estabilidad.
  • Lógica backend: Desarrollada en Python para el control concurrente de nodos.
  • Protocolo: Uso de sockets IP para latencia casi nula.

4. Resultados Clave

  • Latencia Nula: Respuesta instantánea al cambiar de plano.
  • Estabilidad Mecánica: Movimientos fluidos del eje Pan/Tilt gracias al anclaje híbrido.
  • Despliegue Rápido: Nodos totalmente inalámbricos.

5. Futuras Implementaciones

  • Diseño de una Power Distribution Board dedicada para los picos de los motores.
  • Implementación de reglas QoS (Quality of Service) para prioridad de red.
